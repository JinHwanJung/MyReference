# == 과 is 비교.

<code>==</code> : 변수가 참조하고 있는 값에 대해 비교하여 같으면 True를 반환한다.<br/><br/>

# not v and v != 0

- v의 값은 0이 아니다.

    - 값(value)이 0인경우.
    - 객체의 경우 참조(Ref)가 None을 가리키는 경우.
